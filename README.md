Laboratorios de Programación Orientada a Objetos (Java)
Este repositorio contiene una serie de laboratorios y ejercicios prácticos desarrollados para profundizar en los pilares de la POO, utilizando Java 21/24 y buenas prácticas de desarrollo.

 Tecnologías Utilizadas
Lenguaje: Java.

Entorno de Desarrollo: IntelliJ IDEA.

Control de Versiones: Git & GitHub.

 Proyectos Incluidos
1.  Sistema de Gestión de Vehículos
Un ejercicio enfocado en la Herencia y el Polimorfismo.

Conceptos clave: Uso de super(), @Override, y manejo de colecciones con ArrayList.

Funcionalidad: Menú interactivo para registrar Motos y Autos, gestionando atributos específicos como cilindrada y número de puertas.

2.  Sistema de Gestión de Anime (SNK & Pokémon)
Modelado de personajes basado en Shingeki no Kyojin y Pokémon.

Conceptos clave: Encapsulamiento, clases abstractas (lógica) y polimorfismo de datos.

Funcionalidad: Registro dinámico de personajes (Titanes y Pokémon) con atributos especiales como rareza y estado "Shiny".

 Aprendizajes Aplicados
Herencia: Creación de jerarquías de clases para reutilización de código.

Polimorfismo: Tratamiento de diferentes tipos de objetos a través de una clase padre común.

Organización: Estructuración de proyectos mediante packages profesionales.
